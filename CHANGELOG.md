# POPACO 변경 이력

## 2026-04-06

### 기능 추가
- 영수증 첨부 시 카메라 촬영 + 갤러리/파일 이미지 선택 모두 가능
- 저장 시 "저장완료" alert 팝업 표시
- 저장 즉시 Supabase 클라우드 동기화 (await 완료 후 알림)
- 기록 목록에 영수증 카메라 아이콘 표시
- 기록 상세 클릭 시 영수증 사진 바로 보기
- 차량 삭제 시 다른 기기에서도 삭제 동기화 (deletedPlates)
- 다른 기기에서 추가한 차량이 유지되도록 CONFIG 병합 로직 추가

### 변경
- "기사 로그인" → "직원 로그인" 용어 변경
- "기사 이름" → "직원 이름" 용어 변경
- CSV/PDF 보고서 내 "기사" → "직원" 전체 변경
- 불필요한 설정 버튼 제거 (클라우드 동기화, 전체 초기화, 샘플 데이터 재설정)

### 버그 수정
- Supabase RLS(Row Level Security) 비활성화로 데이터 저장 문제 해결
- 클라우드 동기화 시 로컬/클라우드 데이터 병합 (덮어쓰기 방지)
- 클라우드 동기화 에러 로깅 추가
- 5648 테스트 차량 강제 삭제 처리
- 저장 시 클라우드 업로드 완료 후 alert 표시 (await 순서 수정)
